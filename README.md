# 🚀 AI Operations Automation System

## 📌 Overview
AI-powered operations dashboard that automates business analytics and provides real-time insights.  
Built using FastAPI (backend), Streamlit (frontend), and Python data pipelines.

This system replaces manual spreadsheets with automated workflows and intelligent insights.

---

## 🔥 Features
- 📊 Automated ETL pipeline (sales + inventory data)
- 💰 KPI calculations (Revenue, Inventory tracking)
- 🤖 AI-based insights (Top SKU, Low inventory detection)
- 📈 Interactive dashboard with charts and tables
- ⚡ Real-time API-driven architecture

---

## 🧠 Tech Stack
- Python  
- FastAPI (Backend APIs)  
- Streamlit (Frontend Dashboard)  
- Pandas (Data Processing)  
- REST APIs
